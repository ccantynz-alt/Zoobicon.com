Sample media for homepage showcases.

ai-spokesperson-demo.mp4
  Drop a real AI-generated spokesperson clip here (15-30s, muted-safe,
  MP4 H.264, 1280x720 or 1920x1080). The homepage AutoplayVideo
  component (src/components/AutoplayVideo.tsx) will pick it up
  automatically — it autoplays when 40% of the element enters the
  viewport and pauses when it leaves.

Until a real file exists at this path, the component renders a
cinematic animated preview (ambient orbs, pulsing avatar disc,
waveform strip) so the page never looks broken.

Generate the demo clip via zoobicon.com/video-creator once the
pipeline is green.
