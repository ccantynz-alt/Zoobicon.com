== Description ==

**Zoobicon AI** brings powerful AI capabilities to your WordPress site. Generate blog posts, optimize SEO, create images, rewrite content, and translate into 50+ languages — all without leaving the WordPress editor.

= What You Can Do =

* **Content Generation** — Write full blog posts, product descriptions, landing pages, and more from a simple prompt
* **SEO Optimization** — Generate meta titles, descriptions, and keywords. Auto-saves to Yoast SEO
* **Image Generation** — Create AI images and automatically add them to your WordPress media library
* **Alt Text Generation** — Automatically generate descriptive alt text for images (accessibility + SEO)
* **Content Rewriting** — Improve, simplify, expand, shorten, or change the tone of your content
* **Translation** — Translate posts into 50+ languages including Spanish, French, German, Japanese, and more
* **Bulk SEO** — Generate meta tags for multiple posts at once from the posts list

= Works Everywhere =

* **Block Editor (Gutenberg)** — AI sidebar panel right in the editor
* **Classic Editor** — Meta box with one-click AI actions
* **Posts List** — Bulk actions for SEO optimization

= Free & Pro Plans =

* **Free** — 50 AI operations per month. Content generation, SEO, and rewriting included.
* **Pro ($19/month)** — Unlimited operations. Adds image generation, translation, bulk SEO, and priority support.

= Privacy & Security =

* Your content is sent to Zoobicon's API only when you explicitly trigger an action
* No background tracking, no ads, no telemetry
* API key stored securely in your WordPress database
* All requests use HTTPS encryption
* GPL licensed — inspect the code yourself

== Installation ==

1. Upload the `zoobicon-ai` folder to `/wp-content/plugins/`
2. Activate the plugin through the "Plugins" menu
3. Go to **Zoobicon AI** in your admin sidebar
4. Enter your free API key from [zoobicon.com/api-keys](https://zoobicon.com/api-keys)
5. Start using AI in any post or page editor

== Frequently Asked Questions ==

= Do I need an API key? =

Yes, a free API key from zoobicon.com is required. It takes 30 seconds to create one.

= Is it really free? =

Yes. The free plan includes 50 AI operations per month at no cost. No credit card required.

= What AI models power this? =

Zoobicon AI uses Claude (Anthropic), GPT-4o (OpenAI), and Gemini (Google) — the best model is automatically selected for each task.

= Does it work with Yoast SEO? =

Yes. When you generate SEO meta, it automatically saves to Yoast SEO fields if the plugin is installed.

= Is my content private? =

Your content is only sent to the Zoobicon API when you explicitly click "Run AI." It is processed and immediately discarded — we do not store your content.
