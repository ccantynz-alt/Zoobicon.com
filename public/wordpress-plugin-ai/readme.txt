= External Services =

This plugin connects to the Zoobicon AI API (https://zoobicon.com/api/v1/wordpress/) to process AI requests. Data is only sent when you explicitly trigger an action (clicking a button). No data is sent automatically or in the background.

**What is sent:** The content of your post (or selected text) along with the action you requested. For image generation, a text description is sent. For site audit, your site URL is sent.

**When it is sent:** Only when you click "Run AI" or use a bulk action. Never automatically.

**Third-party service links:**

Your content is processed and immediately discarded. We do not store, train on, or share your content.

== Installation ==

1. Upload the `wordpress-plugin-ai` folder to `/wp-content/plugins/`
2. Activate the plugin through the "Plugins" menu
3. Go to **Zoobicon AI** in your admin sidebar
4. Enter your free API key from [zoobicon.com/api-keys](https://zoobicon.com/api-keys)
5. Start using AI in any post or page editor
