# 🚀 Zoobicon

**The AI website builder from the future.**

Zoobicon is a futuristic AI-powered platform that builds stunning websites through intelligent automation. Describe what you want — Zoobicon brings it to life.

---

## ✨ Features

- **AI-Driven Building** — Generate full websites from natural language prompts
- **Futuristic UI** — A sleek, 2040-inspired interface
- **React + TypeScript** — Modern, type-safe component architecture
- **One-Click Deploy** — Ship to production instantly via Vercel

## 🛠 Tech Stack

- **Framework:** Next.js + React
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Deployment:** Vercel
- **AI:** (Coming soon)

## 🚧 Status

## 🏁 Getting Started

```bash
# Clone the repo
git clone https://github.com/ccantynz-alt/Zoobicon.com.git

# Install dependencies
cd Zoobicon.com
npm install

# Run the dev server
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.
